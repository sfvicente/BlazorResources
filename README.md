# Blazor Resources

A collection of useful Blazor resources around the web including blogs, libraries, tools, frameworks, and software.

Blazor is a framework for building interactive client-side web UI with .NET. You can create rich interactive UIs using C# instead of JavaScript, share server-side and client-side app logic written in .NET and render the UI as HTML and CSS for wide browser support, including mobile browsers.
